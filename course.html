About this course

Internet জগতের বহুল জনপ্রিয় প্রশ্নের মধ্যে একটি হলো How to create a website? বাংলাদেশে IT Industry এর উন্নতির সাথে সাথেই বেড়েছে Web Designer এর প্রয়োজনীয়তা। আপনার অথবা আপনার ব্যাবসার একটি অনলাইন প্ল্যাটফর্ম তৈরি করাই ওয়েব ডিজাইনের মুল কাজ।

ওয়েব ডিজাইনে ক্যারিয়ার গড়তে চাইলে প্রথমেই আপনাকে জানতে হবে Web Design এর ভাষা। StudyCenter এর Web design Course-এ আপনি পাবেন HTML:5, CSS, Bootstrap, Javascript এবং React.Js এর উপর একটি সম্পূর্ণ Web design tutorial। যেটি আপনাকে করে তুলবে ওয়েব ডিজাইনে দক্ষ এবং স্বাবলম্বী। Job Market নিজের কারিয়ার গড়তে Enroll করুন Web Design কোর্সে।
